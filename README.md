## 牧师与魔鬼
### 列出游戏中提及的事物
1.Boat 船  
2.River 河  
3.Side of the river 河岸  
4.Three Devils 三个魔鬼  
5.Three Priests 三个牧师  
### 用表格列出玩家动作表（规则表）
帮助3个牧师（方块）和3个魔鬼（圆球）渡河。  
船最多可以载2名游戏角色。  
船上有游戏角色时，你才可以点击这个船，让船移动到对岸。  
当有一侧岸的魔鬼数多余牧师数时（包括船上的魔鬼和牧师），魔鬼就会失去控制，吃掉牧师（如果这一侧没有牧师则不会失败），游戏失败。  
当所有游戏角色都上到对岸时，游戏胜利。
